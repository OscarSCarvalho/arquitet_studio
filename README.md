# Arquitet Studio

Uma experiência digital imersiva para descobrir seu perfil arquitetônico através de um quiz interativo. Este projeto utiliza React, Vite e Tailwind CSS para criar uma jornada personalizada de design de interiores.

## Funcionalidades

- **Onboarding Elegante**: Fluxo suave e acolhedor com micro-interações discretas
- **Discovery Psicológico**: Perguntas step-by-step para captar comportamento e emoção
- **Moodboard Inteligente**: Preview personalizado com cores, texturas e direção criativa

## Tecnologias Utilizadas

- **React 19**: Framework JavaScript para interfaces de usuário
- **Vite**: Ferramenta de build rápida e moderna
- **Tailwind CSS**: Framework CSS utilitário para estilização
- **ESLint**: Ferramenta de linting para manter a qualidade do código

## Como Executar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Abra [http://localhost:5180](http://localhost:5180) no navegador.

## Scripts Disponíveis

- `npm run dev`: Inicia o servidor de desenvolvimento
- `npm run build`: Cria a build de produção
- `npm run lint`: Executa o ESLint para verificar o código
- `npm run preview`: Visualiza a build de produção localmente
